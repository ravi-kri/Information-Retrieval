[Note: These steps are for a windows machine]
Steps to run these files:-

1) Install python 3.6 in your system. It can be downloaded from here:- https://www.python.org/downloads/release/python-360/
2) Install PyCharm. It can be downloaded from here:- 
https://www.jetbrains.com/pycharm/download/#section=windows
Download the Community Version
3) Keep all the python files in "PycharmProjects" in C drive of your system.
4) Open the files using PyCharm and Go to File->Settings->Project Interpreter. Click on the '+' symbol and install the package beautifulsoup4
5) Change the values of variables, HTML_FILES_DIRECTORY and URL_DIRECTORY. Put in a directory that exists on your system and where you would want to save the HTML Raw Files and text file containing all the URLs.
6)Run the file. If asked for an interpreter, choose python 3.6.

Maximum depths:-
1) Task1_BFS - Maximum depth reached is 2
2) Task1_DFS - Maximum depth reached is 6
3) Task_2_Keywords - Maximum depth reached is 2
